== Description ==

Show off numbers or stats on your website using animated Counter block for Gutenberg.

## FEATURES ##
 - Starting Number
 - Ending Number
 - Prefix Text
 - Suffix Text
 - Thousand Separator (delimiter)
 - Animation Speed
 - Title
 - Styling options
